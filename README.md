# تمرینیار هنگبرد (Grip Trainer) 🧗‍♂️💪

برنامه اندروید **تمرینیار هنگبرد** یک مربی شخصی هوشمند، مدرن و سبک‌وزن است که به زبان فارسی و با پشتیبانی کامل از چینش راست‌چین (RTL) طراحی شده تا صخره‌نوردان و ورزشکاران صعود ورزشی را در طول یک برنامه تمرینی ۱۲ هفته‌ای هماهنگ هنگبرد و بارفیکس هدایت کند.

---

## 🎨 نمای کلی طراحی و ویژگی‌های برنامه

طراحی رابط کاربری برنامه با الهام از طبیعت صعود و متریال ۳ (Material Design 3) به صورت مینیمال و کاربرپسند با رنگ‌بندی ورزشی و پرانرژی (نارنجی، سفید و خاکستری تیره) پیاده‌سازی شده است. ویژگی‌های کلیدی آن عبارتند از:

1. **برنامه ۱۲ هفته‌ای علمی و دوره‌ای:**
   * **ماه اول (هفته ۱ تا ۴):** تمرکز روی آماده‌سازی تاندون‌ها و انگشتان با آویزان شدن از گیره‌های بزرگ (۸ ثانیه آویز، ۶۰ ثانیه استراحت) بدون بارفیکس.
   * **ماه دوم (هفته ۵ تا ۸):** تمرکز روی گیره‌های متوسط (۱۲ ثانیه آویز، ۴۵ ثانیه استراحت) به همراه اضافه شدن بارفیکس نیمه به صورت تناوبی در روزهای مختلف.
   * **ماه سوم (هفته ۹ تا ۱۲):** تمرکز روی گیره‌های کوچک (۱۵ ثانیه آویز، ۴۵ ثانیه استراحت) به همراه اضافه شدن ست‌های بارفیکس کامل.
2. **هدایت گام‌به‌گام تمرین (رابط کاربری تعاملی):**
   * نمایش و زمان‌بندی دقیق مراحل گرم‌کردن پیش از تمرین (چرخش گردن، شانه، مچ، باز و بسته کردن انگشتان، آویز سبک).
   * تایمر دایره‌ای معکوس بزرگ و تعاملی برای زمان‌های آویزان شدن و زمان‌های استراحت با امکان پخش/مکث و عبور از هر مرحله.
   * ثبت خودکار درجه سختی تمرین بلافاصله پس از پایان کار با ارائه بازخورد صوتی و تصویری.
3. **تقویم تعاملی تمرینات:**
   * نمایش وضعیت کل جلسات ۳۶ گانه در یک نگاه با رنگ‌های تفکیک شده (سبز برای تمرینات موفق، قرمز برای از دست رفته، خاکستری برای روزهای باقی‌مانده) و ذخیره میزان سختی انتخاب شده.
4. **بخش آمار و پیشرفت شخصی:**
   * محاسبه خودکار درصد پیشرفت کل دوره، تعداد جلسات به اتمام رسیده و ثبت رکوردهای شخصی صعود مانند بیشترین زمان آویزان شدن و بیشترین تکرار بارفیکس.
5. **تنظیمات پیشرفته صخره‌نورد:**
   * امکان سفارشی‌سازی مشخصات فردی (نام، سن، قد و وزن).
   * پیاده‌سازی سیستم مدیریت پوسته (حالت تاریک و روشن پویا).
   * سیستم هشدار و اعلان روزانه با استفاده از `AlarmManager` جهت یادآوری ساعت تمرین با قابلیت تغییر زمان.
6. **کارکرد کاملاً آفلاین و ایمن:**
   * ذخیره‌سازی محلی و ایمن تمامی اطلاعات صخره‌نورد در پایگاه داده محلی گوشی بدون نیاز به اینترنت یا ثبت‌نام.

## 📁 ساختار فایل‌های ایجاد شده در پروژه

پروژه بر اساس معماری استاندارد **MVVM (Model-View-ViewModel)** و با تفکیک وظایف تمیز به شرح زیر سازماندهی شده است:

```
/app/src/main/java/com/example/
│
├── data/                               # لایه داده و ذخیره‌سازی محلی (Room Database)
│   ├── UserProfile.kt                  # مدل اطلاعات صخره‌نورد، تنظیمات ظاهر و رکوردهای تمرینی
│   ├── WorkoutSession.kt               # مدل جلسات تمرینی ۳۶ گانه، وضعیت انجام و درجه سختی
│   ├── WorkoutDao.kt                   # واسط دسترسی به داده‌ها (Queries) برای جدول‌ها
│   ├── AppDatabase.kt                  # کلاس راه‌اندازی و ایجاد پایگاه داده محلی SQLite Room
│   └── WorkoutRepository.kt            # مخزن داده برای هماهنگ‌سازی لایه‌های مختلف و مقداردهی اولیه داده‌ها
│
├── receiver/                           # لایه گیرنده‌های سیستم اندروید
│   └── NotificationReceiver.kt         # دریافت سیگنال از AlarmManager جهت نمایش اعلان روزانه در زمان مقرر
│
├── ui/                                 # لایه رابط کاربری (Jetpack Compose)
│   ├── theme/                          # تم‌ها، رنگ‌ها و فونت‌های شخصی‌سازی شده برنامه
│   │   ├── Color.kt                    # پالت رنگی گرم متناسب با سنگ‌نوردی (نارنجی و خاکستری تیره)
│   │   ├── Theme.kt                    # مدیریت تم روشن/تاریک بر اساس تنظیمات انتخابی کاربر
│   │   └── Type.kt                     # تنظیمات تایپوگرافی متن‌ها
│   │
│   ├── screens/                        # صفحات مختلف برنامه (Screens)
│   │   ├── HomeScreen.kt               # صفحه اصلی شامل خوش‌آمدگویی، خلاصه پیشرفت، تمرین امروز و تقویم ۱۲ هفته‌ای
│   │   ├── WorkoutScreen.kt            # صفحه فعال تمرین شامل گرم‌کردن، تایمر هوشمند آویزان شدن/استراحت و نظرسنجی سختی
│   │   ├── ProgressScreen.kt           # صفحه گزارشات شامل رکوردهای شخصی، تعداد جلسات و تاریخچه تفصیلی تمرینات
│   │   └── SettingsScreen.kt           # تنظیمات کاربری شامل مدیریت مشخصات، تغییر تم، زمان‌بندی یادآور روزانه و پاکسازی داده‌ها
│   │
│   └── MainViewModel.kt                # مدیریت وضعیت کل برنامه، جریان تایمرها، گام‌های تمرینی و پردازش اطلاعات پس‌زمینه
│
├── util/                               # ابزارهای کمکی پروژه
│   └── NotificationHelper.kt           # راه‌اندازی کانال‌های نوتیفیکیشن و زمان‌بندی دقیق با AlarmManager
│
└── MainActivity.kt                     # نقطه ورود اصلی برنامه، راه‌اندازی Edge-to-Edge و نوارهای ناوبری پایین صفحه (NavigationBar)
```

## 🛠️ تکنولوژی‌ها و کتابخانه‌های کلیدی مورد استفاده

* **Kotlin:** زبان برنامه‌نویسی اصلی پروژه با ایمنی بالا و هماهنگی کامل با کدهای مدرن.
* **Jetpack Compose:** طراحی کل رابط کاربری به صورت کاملاً اعلانی (Declarative) و مدرن بدون استفاده از XML های سنگین.
* **Room Database:** لایه انتزاعی قدرتمند روی SQLite برای ذخیره محلی و ایمن اطلاعات کاربران صخره‌نورد.
* **Coroutines & Flow:** مدیریت جریان‌های داده‌ای غیرهمزمان به صورت کاملاً غیرمسدودکننده (Non-blocking).
* **AlarmManager & Notifications:** پیاده‌سازی زنگ‌های دقیق محلی برای بیدارباش و ارسال اعلان‌های منظم روزانه جهت یادآوری تمرین‌ها.
* **Material Design 3 (M3):** استفاده از آخرین استانداردهای طراحی متریال گوگل برای المان‌ها، دکمه‌ها، کارت‌ها و منوهای متحرک.
